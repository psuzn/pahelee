pahelee
===============================

version number: 0.0.1
author: Sujan Poudel

Overview
--------

Maze generator using recursive depth first search algorithm

Installation / Usage
--------------------

To install use pip:

    $ pip install pahelee


Or clone the repo:

    $ git clone https://github.com/psuzn/pahelee.git
    $ python setup.py install
